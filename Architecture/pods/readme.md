#### Replicasets
- have more than one pods to provide availability when one pod dies. 
- ReplicationController helps in managing pods and replicates based on the desired date. 
- used to load balance and scale, the pods can be deployed in the same node or in a different node. 
- Replica sets replaces the replication controller.
